## Lab Overview

### **Lab 01 – Basic Router Configuration**
Learn fundamental router setup, device addressing, and basic CLI commands.

### **Lab 02 – DNS and ARP**
Understand domain resolution, ARP mechanics, and packet flow analysis.

### **Lab 03 – Identifying Issues over Network & Solutions**
Explore common networking problems and learn diagnostic approaches.

### **Lab 04 – Troubleshooting Connectivity Issues**
Hands-on troubleshooting using real-world scenarios and tools.

### **Lab 05 – Configuring IPv4 Static & Default Routes**
Configure static routing, default routes, and validate connectivity paths.

## Additional Sections
